# crud_aws_demo
# CRUD de Usuários em Python na AWS EC2

Este projeto consiste em uma aplicação simples de CRUD (Create, Read, Update, Delete) desenvolvida em Python e executada em uma instância Linux na AWS EC2, acessada remotamente via SSH.

## Tecnologias utilizadas
- Python 3
- Linux (Ubuntu / Debian)
- AWS EC2
- SSH
- Git e GitHub

## Funcionalidades
- Criar usuários
- Listar usuários
- Atualizar usuários
- Remover usuários
